algorithms
=============
A collection of algorithms written in C++.

Algorithms
---------------------

* collatz - Provides a forward iterator over the sequence produced by the
            Collatz conjecture.
* fibonacci - Provided a forward iterator over the Fibonacci sequence.

Building
---------------------
On Windows:
`$ make CXX=cl`

On Linux:
`$ make`
